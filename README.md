# 🤖 AI Decision-Making Agent (LangGraph + LLM)

An AI-powered decision-making system built using LangGraph that dynamically routes user queries and performs intelligent actions.

## 🌐 Live Demo
https://ashish--aiagent.streamlit.app/

---

## 🚀 Features

- Intelligent decision-making agent
- Dynamic routing of tasks
- Multi-step reasoning using LangGraph workflows
- Open AI LLM powered responses
- Deployed interactive web app

---

## 🧠 How It Works

1. **User Input**
   - User submits a query

2. **Decision Node**
   - LLM analyzes query and determines action

3. **Graph Execution**
   - Query is routed through LangGraph nodes

4. **State Management**
   - Context is maintained across steps

5. **Response Generation**
   - Final response is generated after reasoning steps
